## install graphql-server dependencies
```
cd graphql-server
```

```
pnpm install
```

## Execute the server:
```
npx nodemon index.js
```

## Execute the DB:
```
pnpm run json-server
```

## install react-graphql dependencies
```
cd react-graphql
```

```
pnpm install
```

## Execute frontend:
```
pnpm run dev
```
